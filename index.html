About this project

This is a plain static website made with HTML, CSS and a few lines of JavaScript for the mobile menu. It requires no build process and can be opened locally or published directly with GitHub Pages.

Cost reminder: Always complete the cleanup section after a lab. Load balancers, EC2 instances, EBS volumes, Elastic IP addresses and NAT gateways may generate charges.
