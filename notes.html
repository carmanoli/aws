CloudFormation

Avoid deletion of Critical Resources:

How to protect databases (RDS) or storage buckets (S3) from being accidentally destroyed when a Stack is deleted? The answer involves setting the DeletionPolicy attribute: Retain

CloudFormation Drift Detection:

How to detect when the actual state of resources in your AWS environment deviates from the expected state defined in your CloudFormation template? The answer involves using the Drift Detection feature to compare the current state with the expected state and identify any discrepancies.
